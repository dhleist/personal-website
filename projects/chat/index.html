chat

I noticed that, at my univeristy, students in the same class typically use a generic chat application in order to communicate about upcoming assignments and exams. Although it works, I thought additional functionality could improve the efficacy and overall experience. Realizing this, I started to create a dedicated service that I hope to complete soon.

I decided to use React for the front end to gain experience using the library, and Java for the back end to further develop my skills in the language. As soon as it's complete, I plan on releasing the initial version so everyone can try it out!
