to-do-list

There are many to-do list applications out there, but I never found one that was exactly what I wanted in terms of aesthetic and functionality. So, I built my own in order to effectively keep track of my day-to-day tasks.

Constructed using only HTML, CSS, and vanilla JavaScript, it uses web storage instead of a database to save the items entered by the user. Check out the live version and try it out for yourself!
